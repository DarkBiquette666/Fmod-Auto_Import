v0.5.0 (December 2024)
- Major core architecture refactoring for maintainability
- Extracted project.py (1,137 lines) into 9 specialized core modules (186 lines)
- 84% code reduction in main project facade
- New core/ directory with specialized managers:
  - xml_writer.py: Shared XML formatting utilities
  - xml_loader.py: Centralized XML parsing
  - pending_folder_manager.py: Transaction staging system
  - event_folder_manager.py: Event folder CRUD operations
  - asset_folder_manager.py: Asset folder CRUD operations
  - bus_manager.py: Bus CRUD operations
  - bank_manager.py: Bank CRUD operations
  - event_creator.py: Event template cloning
  - audio_file_manager.py: Audio file XML entries
- Stateless manager pattern with facade delegation
- 100% backward compatibility maintained
- Follows SOLID principles (Single Responsibility)
- All modules < 400 lines (highly maintainable)
