FMOD Moco Auto Import - Project Structure
==========================================

Fmod Scripts/
├── moco_auto_import.py              # Main Python GUI application (139KB)
├── launch_moco.bat                  # Quick launcher for the GUI
├── README.md                        # Full documentation
├── .gitignore                       # Git ignore rules
│
├── Script/
│   └── _Internal/
│       └── MocoImportFromJson.js    # FMOD Studio import script (21KB)
│
└── Moco AutoImport/                 # Test FMOD project
    └── MocoAutoImport/
        ├── MocoAutoImport.fspro     # FMOD project file
        ├── Assets/                  # Audio assets folder
        ├── Metadata/                # FMOD project metadata
        ├── Build/                   # Built banks (ignored by git)
        ├── .cache/                  # FMOD cache (ignored by git)
        └── .user/                   # User settings (ignored by git)

Essential Files:
- moco_auto_import.py: Main application with GUI and import logic
- Script/_Internal/MocoImportFromJson.js: FMOD API integration
- launch_moco.bat: Convenience launcher

Temporary Files (auto-generated, ignored by git):
- moco_import_*.json: Import payload
- _temp_import_wrapper.js: Wrapper script for fmodstudiocl.exe
- import_result_*.json: Import results
- import_result_*_debug.txt: Debug logs

User Settings (saved to home directory):
- ~/.moco_auto_import_settings.json: Persistent settings
